Crear un servido o trabajar con node

Comandos a usar:

  • npm init, donde se guardan las depencencias
  • npm i nodemon, dependencia a instalar(la dieron en el desafio)

dentro del package.json creo el script: "scripts": { "start": "nodemon ./servidorWeb/servidor.js", } desp de eso ejecuto el comando: npm run start

nodemon nos permite actualizar de forma automatica la consola en visual o reenlanzar

Que es un servidor web:

dispositivo que realiza conexiones remotas a redes para recibir o trasferir datos. Para esto se requiere un puente de datos llamado HTTP.

Que es express

Express nos permite definir, para cada tipo de peticion HTTP que llegue a una determinada URL, que acciones debe tomar, mediante la deficion de un callback para cada caso que condideremos necesario incluir en nuestra API
